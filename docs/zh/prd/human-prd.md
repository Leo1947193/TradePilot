# 背景

我是一个美股新手，我买卖股票的工作流程如下：
1. 刷小红书，了解当前热点和可能影响市场的事件
2. **主观地**选择我认为能盈利的股票
3. 做一些基础的技术分析：只看K线图、MACD、RSI，判断价格会涨还是会跌

# 我的问题

个人投资者在做股票买入决策时，通常面临以下挑战：

1. **缺乏结构化的分析流程**

    投资决策往往依赖主观判断，而非对基本面因素（如财务状况、行业背景）和技术指标（如趋势、成交量、支撑/阻力位）的系统性评估。

2. **信息碎片化，缺乏整合**

    现有工具（如行情数据平台和图表工具）主要提供原始数据和指标，但无法有效地将信息整合为可操作的洞察。用户只能手动解读并综合各种分散的信号。

3. **缺乏客观的决策支持**

    投资者难以将基本面、技术面、市场情绪等多维度信号整合为统一、可比较、客观的决策依据。


**因此**，需要一个股票决策支持系统，能自动化分析并提供结构化输出，以：

- 减少分析时间和精力

- 提高决策的客观性

- 支持投资策略的持续回顾与迭代改进

# 期望工作流程
1. 用户输入自己选择的股票
2. 系统开始研究
3. 向用户输出包含**技术分析**、**轻量级基本面分析**、**情绪与预期**、**事件驱动分析**的结构化报告，以及**交易建议（如不同的买入和卖出时机建议，并附上各自的风险）**，分析中应包含具体的信息来源
4. 我决定是否采纳建议
